FLEX

Description

  • Flex utility can be used to manage the layout, alignment, grid columns, navigation and other components of the page.
  • It makes easy to design layout structure without using float or positioning.

Flex Behaviors and Direction

    Flex Behaviors

  • The flexbox display utilities like d-flex and d-inline-flex to display the flexbox container and children elements into flex items.
